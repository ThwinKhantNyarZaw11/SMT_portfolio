E-Commerce UI Design

Project Overview

Designed a responsive, mobile-first e-commerce interface for a fashion brand to enhance user engagement and optimize conversions. Focused on streamlining browsing, refining product discovery, and minimizing cart abandonment.

Key Highlights

  • Crafted UI for homepage, product listings, product detail pages, and checkout
  • Implemented sticky CTAs, live stock indicators, and smart filters
  • Designed reusable card components and microinteractions for better UX
  • Used real user feedback to iterate on flow and button placements
  • Optimized layout for thumb zones and one-handed use on mobile

Tools Used

  • Figma – UI design, component library, and mobile breakpoints
  • Hotjar – User session recording and heatmaps
  • Google Optimize – A/B testing layouts
  • Zeplin – Developer handoff and redlines

Impact

  • Increased mobile checkout conversion by 22%
  • Reduced bounce rate by 15% on product detail pages
  • Enabled faster dev handoff with clean design tokens and guidelines
